Narcissus css bootstrap

Introduction

Narcissus is a minimal css bootstrap for web-based projects that tries to diverge as little as possible from the standard rendering of a webpage without css.

Some notable divergents are:

  • Headers are centered
  • Headers are all uppercase
  • Headers have a normal font weight
  • Buttons are completely rendered with css
